  * новый маркер {{turret}}:
    Отображается, если на танке сток башня.
	Символ "*" - сток башня. Невозможно поставить топ орудие. Субъект сильно уязвим.
	Символ "-" - сток башня. Топ орудие возможно. Субъект немного уязвим.
	Сток башня может быть слабее бронирована, иметь меньший обзор или медленный поворот.
	Кроме того, на танке со сток башней экипаж может быть слабо прокачан.
  * отдельные настройки для всех элементов во всех состояниях
  * адаптация к WoT 0.8.0

XVM-2.5.2:
  * переработаны иконки кланов. Сейчас в конфиге только один параметр:
    battle/clanIconsFolder, в котором указан путь к корневой папке иконок
    кланов. Все иконки загружаются автоматически из подпапки с названием,
    соответствующем региону игры (RU, EU, US, и т.д.).
    Чтобы добавить иконку клана или игрока, просто скопируйте файл иконки в:
      clanicons/[REGION]/clan/ (для клана)
      clanicons/[REGION]/nick/ (для отдельного игрока)
    Так же можно сделать иконку по умолчанию для клана или игрока:
      clanicons/[REGION]/clan/default.png (для клана по умолчанию)
      clanicons/[REGION]/nick/default.png (для игрока по умолчанию)
    Иконки ищутся в следующем порядке:
      nick/<player>.png -> clan/<clan>.png -> clan/default.png -> nick/default.png
  * добавлен параметр конфига:
      battle/clanIconsFolder - корневая папка иконок кланов
  * удалена секция конфига:
      players

  * добавлены архивы с полными наборами иконок кланов:
      http://code.google.com/p/wot-xvm/downloads/detail?name=clanicons-full-ru.zip (RU)
      http://code.google.com/p/wot-xvm/downloads/detail?name=clanicons-full-eu.zip (EU)
      http://code.google.com/p/wot-xvm/downloads/detail?name=clanicons-full-us.zip (US)
  * удалена сетка координат (параметр battle.drawGrid)

  * добавлен макрос {{rlevel}} для уровня танка римскими цифрами
  * динамический цвет и прозрачность для тени
  * добавлен параметр font/italic (значения: true/false)
  * добавлен макрос {{c:vtype}} и секция colors/vtype для динамического цвета
    по типу техники
  * добавлен цвет для премиумной техники в макрос {{c:vtype}}
